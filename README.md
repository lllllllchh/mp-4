Harvard Art Search

A simple Next.js web app that lets users search artworks by artist using the Harvard Art Museums API. The project demonstrates server-side API fetching, reusable components, and clean UI organization.

**Features**

 Search artworks by artist name

 Displays artwork images, titles, dates, and classifications

 Server-side API fetching using a secure API route (no exposed key)

 Reusable ArtCard component for each artwork

 Built with Next.js 14 App Router + Server Components
